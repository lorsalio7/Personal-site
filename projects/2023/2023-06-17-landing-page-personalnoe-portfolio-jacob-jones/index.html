Landing page персональное портфолио Jacob Jones

#верстка / 17.06.2023

Портфолио

Верстка лендинга портфолио с полноэкранными секциями. В работе использовалась svg графика, в секциях с карточками был использован css calc() чтобы карточки были резиновыми. В секции «Service» был использован аккордион, а в секции «Testimonial» был использован слайдер.

Посмотреть

Используемые плагины:

  • Splide - библиотека слайдера. Автор - Naotoshi Fujita
  • Smooth Scroll - библиотека плавного скролла по клику на ссылки с якорями. Автор - Chris Ferdinandi
