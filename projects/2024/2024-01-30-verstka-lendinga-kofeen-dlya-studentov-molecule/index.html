Верстка лендинга Molecule

#верстка / 30.01.2024

Портфолио

Верстка лендинга кофеен для студентов «Molecule». Макет отличался сложной секцией Hero, которая была реализована с помощью Grid CSS. В работе применялись активно CSS переменные. Также секция с картами, были использованы кастомные кнопки зума Яндекс Карт. Также добавлен анимированный прелоадер.
